Why Rust

Zero-cost abstractions과
메모리 안전성

Rust는 가비지 컬렉터 없이 메모리 안전성을 보장합니다. 소유권 시스템 덕분에 컴파일 타임에 대부분의 버그를 잡아냅니다. 

use clap::Parser;

#[derive(Parser)]
#[command(name = "rtk", about = "Rust Token Killer")]
struct Cli {
    #[arg(short, long)]
    verbose: bool,

    #[command(subcommand)]
    command: Commands,
}
